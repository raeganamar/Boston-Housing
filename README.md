# Boston-Housing
# 🏠 Boston Housing Price Prediction — Day 33

---

> ## 📦 Project Summary
>
> **Background**
> Harga properti dipengaruhi oleh berbagai faktor seperti kondisi lingkungan, fasilitas, dan karakteristik wilayah. Penentuan harga yang tidak akurat dapat menyebabkan kerugian bagi buyer maupun seller.
>
> **Objective**
> Membangun model machine learning untuk memprediksi harga rumah menggunakan dataset Boston Housing serta memahami faktor-faktor yang paling berpengaruh terhadap harga.
>
> **Approach**
>
> * Melakukan exploratory data analysis (EDA) untuk memahami pola data
> * Membangun model regresi (Linear Regression) sebagai baseline
> * Menggunakan teknik regularization (Ridge & Lasso) untuk mengatasi overfitting
> * Evaluasi model menggunakan metrik regresi (RMSE, R²)
>
> **Key Findings**
>
> * Fitur seperti jumlah kamar (**RM**) berpengaruh positif terhadap harga rumah
> * Faktor sosial seperti **LSTAT** memiliki korelasi negatif terhadap harga
> * Regularization (Ridge & Lasso) membantu meningkatkan generalisasi model
>
> **Business Value**
> Model dapat digunakan sebagai baseline untuk estimasi harga properti, membantu agen dan investor dalam menentukan pricing strategy serta mengidentifikasi area dengan potensi investasi tinggi.
>
> **Conclusion**
> Pendekatan regresi dengan regularization berhasil meningkatkan stabilitas model dan memberikan insight yang relevan terhadap faktor penentu harga rumah.
>
> 📈 *Next Improvement:* Hyperparameter tuning & feature selection lanjutan untuk meningkatkan performa model.
